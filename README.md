# Tonaquelove — Love & Resilience blog (static starter)

This static starter focuses on true love and practical guides for staying together through difficult moments.

Included
- index.html — homepage
- about.html — mission and editorial focus
- posts/ — posts including welcome, definition-of-true-love, and staying-together
- assets/styles.css — styles

Preview locally
1. Copy files into your repository.
2. Run a static server, for example:
   - Python 3: `python -m http.server 8000`
   - Visit: `http://localhost:8000`

Publishing with GitHub Pages
1. Add files to a branch (commonly `gh-pages` or `main`).
2. In GitHub: Settings → Pages:
   - Source: Branch: `gh-pages` (or `main`)
   - Folder: `/` (root)
3. Save and wait for the site to publish:
   - URL: `https://tmarzz-cmyk.github.io/Tonaquelove/` (or custom domain)
4. (Optional) Add CNAME for a custom domain and configure DNS, then enable "Enforce HTTPS".
